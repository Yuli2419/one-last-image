# 🧸「One Last Image」One Last Kiss 风格封面生成器

One Last Image 是一个 将 **赛璐璐风格** **动画截图** 或 **插画** 转换成 One Last Kiss 封面风格的在线生成器

## 地址
https://lab.magiconch.com/one-last-image/

在转换时，可以自定义
 - 线条处理方案
 - 开关 One Last Kiss 风格（仅将图片转换为线稿）
 - 给画面暗部排铅笔调子
 - 叠加类似 One Last Kiss 光碟封面水印
 - 初回限定盘面效果选项
 - 线迹轻重
 - 调子数量
 - 叠加歌词 （未完成）

点按图片可以和原图对比生成效果

手机端请使用自带浏览器进行保存
